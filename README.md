FSharp-Maze
===========

Maze generator using F#.

จงแสดงว่าคุณจะเก็บข้อมูลเขาวงกตนี้อย่างไร โดยเป็น grid สองมิติ มีส่วนที่เดินได้ และเดินไม่ได้ มีจุดเริ่ม(Start) และจุดสิ้นสุด(Finish) ให้แสดงเป็นโค้ดสั้นๆ หรือ UML Class diagram
